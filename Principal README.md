# 📘 Repositorio Matemáticas Primaria

Bienvenido al Repositorio de Matemáticas para Básica Primaria 
Un espacio diseñado para fortalecer el aprendizaje matemático de manera dinámica, creativa y significativa.
Aquí encontrarás recursos interactivos, juegos, guías, videos y materiales didácticos para estudiantes de 1° a 3° de primaria.

## 🎯 Objetivo
Se busca brindar recursos accesibles, dinámicos y contextualizados que fortalezcan el pensamiento matemático y promuevan un aprendizaje significativo desde los primeros años escolares.
